Technical stuff

If you're interested in the technical behind this blog, I'm using Pelican. It's a static site generator, which means unlike full-fledged blogging frameworks like Wordpress or Ghost, the static html gets generated from templates and then uploaded to be hosted.

Github provides free hosting and the domain is my own.
