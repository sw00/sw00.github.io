Docker for Windows Users Part I

I've seen a lot of developers either unaware and confused as to what exactly Docker and containerisation is all about. This is because they have either never encountered the use case for it, or primarily work in Windows environments. With these blog series, I'll try and explain what all the fuss is about and hopefully how you can use it in your little way.

Contributing to the Pixelated Project

I had the opportunity to participate in a hack night for the Pixelated Project.
